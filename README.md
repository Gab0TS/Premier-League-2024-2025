# Premier League 2024-2025

Este proyecto muestra las principales estadísticas, tanto de equipos como jugadores en la temporada 2024/2025 de la Premier League. 
Para ello se despliega un menú que muestra las opciones que el usuario puede elegir:

## Player_Stats
1. Máximos goleadores (5)
2. Máximos asistidores (5)
3. Mayor cantidad de tarjetas amarillas (5)
4. Mayor cantidad de tarjetas rojas (5)
5. Nacionalidades más frecuentes (5)
6. Jugadores de una determinada nacionalidad (lista)

## Team_Stats
1. Mayor cantidad de goles anotados por jugadores(5)
2. Mayor cantidad de asistencias dadas por jugadores(5)
3. Mayor cantidad de offsides (5)
4. Mayor cantidad de tarjetas amarillas (5)
5. Mayor cantidad de tarjetas rojas (5)
6. Equipos con mayor cantidad de jugadores de una determinada nacionalidad (5)
7. Nacionalidades de un determinado equipo (5)
